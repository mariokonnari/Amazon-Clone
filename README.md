# Mariozon 🛒

An Amazon-inspired e-commerce web application built with React, TypeScript, Tailwind CSS and Redux Toolkit.

🔗 **Live Demo:** https://mariozon.vercel.app/

## Features
- Browse real products from the Fake Store API
- Product detail pages with related products
- Real-time search filtering
- Shopping cart with quantity controls
- Persistent cart state with Redux
- Fully responsive design

## Tech Stack
- React + TypeScript
- Tailwind CSS v4
- Redux Toolkit
- React Router v6
- Vite
- Deployed on Vercel

## Getting Started
git clone https://github.com/mariokonnari/Amazon-Clone  
cd Amazon-Clone  
npm install  
npm run dev
